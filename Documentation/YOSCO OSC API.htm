YOSCO OSC API v 1.0.2

INPUT

ADDRESSFORMATDATAEXAMPLEDESCRIPTION
/set/mixer/input/[n]/gain,i[-96 to 66]

/set/mixer/input/1/gain -6

Sets the HA gain of an input channel.
/set/mixer/input/[n]/fader/level,f[-138. to 10.]

/set/mixer/input/2/fader/level -12.5

Sets the fader level of an input channel.
/set/mixer/input/[n]/fader/on,i or ,s[1,0],[on,off]

/set/mixer/input/4/fader/on 0

/set/mixer/input/4/fader/on off

Mutes or unmutes channel strip; 1 = 'ON', 0 = 'OFF'.
/set/mixer/input/[n]/pan,i[-63 to 63]

/set/mixer/input/4/pan 15

Sets the pan of an input channel; -63 is hard left, 63 is hard right.
/set/mixer/input/[n]/label/name,salphanumeric

/set/mixer/input/4/fader/label Guitar

Sets the text of an input channel's scribble strip. Anything over 8 characters will be truncated.
/set/mixer/input/[n]/label/color,s[Blue, Orange, Yellow, Purple, Cyan, Magenta, Red, Green, Off]

/set/mixer/input/4/fader/label Magenta

Sets the color of an input channel's scribble strip.
/set/mixer/input/[n]/to_mix/[m]/on,i or ,s[1,0],[on,off]

/set/mixer/input/4/to_mix/12/on 1

/set/mixer/input/4/to_mix/12/on off

Mutes or unmutes an input channel's send to an aux mix.
/set/mixer/input/[n]/to_mix/[m]/level,f[1,0],[on,off]

/set/mixer/input/4/to_mix/12/level -12

Sets the send level of an input channel to an aux mix.
/set/mixer/input/[n]/dca/[d]/assign,i[1,0]

/set/mixer/input/4/dca/8/assign 1

Assigns or unassigns an input channel to a DCA.

BRIDGE CONTROL

ADDRESSFORMATDATAEXAMPLEDESCRIPTION
/bridge/network/ip,sIP Address

/bridge/network/ip 192.168.7.10

Changes the IP address that the bridge sends to; set this to the IP of your target console.
/bridge/network/port,i[1024-65535]

/bridge/network/port 35001

Changes the port that the bridge listens on.
/bridge/console,s[QL1, QL5, CL1, CL3, CL5, TF1, TF3, TF5]

/bridge/console CL5

Changes the console model that the bridge translates for.
/bridge/onn/anone

/bridge/on

Activates the bridge.
/bridge/offn/anone

/bridge/off

Deactivates the bridge.
